# Bookstore
중앙문고(2021.03~2021.04) - 교보문고를 모티브로 만든 프로젝트 


## 주요 기능
1. 사용자 페이지

  + 메인 배너 노출
  + 장바구니
  + 카테고리별 상품 노출
  + 상품검색
  + 구매내역
  
 2. 관리자 페이지
  + 주문List 관리
  + 회원 정보 관리
  + 상품등록, 수정, 재고 관리
